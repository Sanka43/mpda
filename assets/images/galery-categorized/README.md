# MPDA Gallery Categorization

Original images remain unchanged in `assets/images/galery`.

Use these categorized copies for the website:

- `01-featured-website` - Best display images for homepage, hero sections, and main gallery highlights.
- `02-cultural-ceremony` - Kandyan/cultural ceremony and ritual/event close-up images.
- `03-dance-classes-training` - Class practice, training sessions, rehearsal, and student activity images.
- `04-outdoor-group-showcase` - Outdoor group formations, temple/background shots, and large group presentation images.
- `05-stage-performances` - Stage performance/event images.
- `06-awards-and-details` - Awards, trophies, props, and detail shots.
- `07-duplicates-review` - Duplicate or near-duplicate files kept separately for review before website use.

Recommendation:

Use `01-featured-website` first for the public-facing gallery. Use `02-cultural-ceremony` and `04-outdoor-group-showcase` as supporting gallery sections. Use `03-dance-classes-training` for academy/classes pages.
